# 📊 Sales Performance Analysis Using Python and Power BI

## 📌 Project Overview

This project was completed as part of the **ApexPlanet Data Analytics Internship (Task 4)**.

The objective of this project is to analyze an e-commerce sales dataset using Python and Power BI to identify customer purchasing behavior, sales trends, product performance, and provide data-driven business recommendations.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Identify top-performing products and cities.
- Study monthly sales trends.
- Perform statistical validation using an Independent Samples T-Test.
- Create an interactive Power BI dashboard.
- Provide business recommendations based on the analysis.

---

## 🧹 Data Preprocessing

The following preprocessing steps were performed:

- Checked for missing values.
- Filled missing values in the Age column using the median.
- Filled missing values in the City column using the mode.
- Verified that no missing values remained.

---

## 📊 Statistical Validation

An Independent Samples T-Test was performed to determine whether there was a significant difference in spending between male and female customers.

**Null Hypothesis (H₀):**
There is no significant difference in average spending between male and female customers.

**Alternative Hypothesis (H₁):**
There is a significant difference in average spending between male and female customers.

## 💡 Business Recommendations

- Increase inventory for high-demand products.
- Focus marketing campaigns on high-performing cities.
- Introduce customer loyalty programs.
- Offer seasonal promotions during low-sales periods.
